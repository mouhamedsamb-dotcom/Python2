# Python2
## Janken (Pedra, Paper i Tisora)
El jugador selecciona una opció i la màquina en tria una de manera aleatòria en un fitxer que mos han proporcionat els professors.

Regles:

-Pedra guanya a tisora
-Tisora guanya a paper
-Paper guanya a pedra

El jugador podra triar si jugar al millor de 3 o a 5 rondes i al acabar el programa mostra el resultat de la partida (victòria, derrota o empat).

## Adivina el numero 
Juego sencillo en **Python** donde el jugador debe adivinar un número secreto.

### Como funciona
-  Eliges una dificultad
- El programa  genera un numero aleatorio
- Introduces  numeros hasta acertar o quedarte sin intentos
- El juego indica si el numero es mayor o menor
